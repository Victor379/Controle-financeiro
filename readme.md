# 💰 Controle Financeiro Pessoal

Um site simples para controle de renda e gastos mensais.

## 🚀 Funcionalidades

- Registro de entradas (salário, extras, etc.)
- Registro de despesas
- Cálculo automático:
  - Total recebido
  - Total gasto
  - Saldo
- Filtro por mês
- Geração de planilha mensal (CSV)

## 🖥️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📊 Como usar

1. Adicione suas entradas e gastos
2. Acompanhe o saldo no painel
3. Gere sua planilha mensal quando quiser

## 📁 Como rodar o projeto

Basta abrir o arquivo `index.html` no navegador.

## 🌐 Deploy

O projeto pode ser hospedado usando GitHub Pages.
